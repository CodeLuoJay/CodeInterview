JDK、JRE、JVM之间区别和联系

JDK是Java Development Kit （Java开发工具包）是用来执行编译Java源代码文件、解释和执行Java字节码

JRE是Java Runtime Environment（Java运行环境）是用于解释执行Java的字节码文件

JVM是Java Virtual Mechinal（Java虚拟机），是JRE的一部分,它用来运行java字节码文件的虚拟计算机

它们之间的联系可以理解为它们之间的包含关系：JDK>JRE>JVM

即JDK包含JRE,JRE包含JVM,所以总的来说JDK包含了JRE和JVM
